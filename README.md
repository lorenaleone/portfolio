# Lorena Leone — Portfolio

Personal portfolio website. Bilingual (PT/EN), static HTML, hosted on GitHub Pages.

## Stack
- HTML5 + CSS3 (no framework)
- Vanilla CSS with custom properties
- Google Fonts (Fraunces + Inter Tight + JetBrains Mono)
- ~0 JS dependencies

## Structure
```
portfolio/
├── index.html          # PT version (default)
├── en/
│   └── index.html      # EN version
├── assets/
│   └── style.css       # Main stylesheet
└── README.md
```

## Live
- PT: https://lorenaleone.github.io/portfolio
- EN: https://lorenaleone.github.io/portfolio/en

## Deploy
Push to `main` branch. GitHub Pages serves from root.
